若直接使用std::cin >> str读取字符串，会跳过连续的空白符（空格、换行），因此建议使用std::getline(std::cin, str)读取一行字符串，并通过其返回值（bool(std::cin)）判断是否读取到文件末尾。如下所示：

#include <iostream>
#include <string>

int main() {
  std::string line;
  while (std::getline(std::cin, line)) {
    // 处理line
    std::cout << line << std::endl;
  }
}
