```
1. cin.clear()：

作用：清除cin流的错误状态。当输入操作发生错误（例如，期望输入整数却输入了字符）时，cin会设置错误状态位，导致后续的输入操作被跳过。使用cin.clear()可以清除这些错误状态，使流恢复为有效状态，从而可以继续读取输入。

例如，如果输入了一个非数字字符，而程序期望读取整数，那么cin会进入错误状态，此时使用cin.clear()可以清除错误状态，然后才能继续读取。

2. cin.ignore(n, delim)：

作用：从输入流中忽略（丢弃）字符，直到丢弃了n个字符或者遇到了分隔符delim（会丢弃分隔符）为止。

参数：
n：要忽略的字符数量的最大值。
delim：分隔符，当遇到这个字符时，忽略操作停止（包括这个分隔符）。

常见用法：在清除错误状态后，忽略当前行剩余的所有字符，以防止这些字符影响后续的输入。例如，cin.ignore(1000, '\n')会忽略当前行最多1000个字符，直到遇到换行符为止。
```

# 文件操作
```
#include <iostream>
#include <fstream>
#include <string>

using namespace std;

// 写入文件
void writeFile() {
    ofstream outFile("example.txt");
    if (outFile.is_open()) {
        outFile << "Hello, World!\n";
        outFile << "这是第二行文本\n";
        outFile.close();
        cout << "文件写入成功" << endl;
    } else {
        cout << "无法打开文件" << endl;
    }
}

// 读取文件
void readFile() {
    ifstream inFile("example.txt");
    string line;
    if (inFile.is_open()) {
        while (getline(inFile, line)) {
            cout << line << endl;
        }
        inFile.close();
    } else {
        cout << "无法打开文件" << endl;
    }
}

// 追加写入
void appendFile() {
    ofstream outFile("example.txt", ios::app);
    if (outFile.is_open()) {
        outFile << "这是追加的内容\n";
        outFile.close();
    }
}
```
