## 使用哑节点
```
struct Node {
    int val;
    Node* next;
};

// 创建哑节点
Node dummy(0);  // 值可以是任意值，通常用0
// 或者
Node *tail=&dummy
```
```
Node* addNode(Node* head, int value) {
    Node dummy(0);
    dummy.next = head;  // 哑节点指向原链表
    
    Node* tail = &dummy;
    while (tail->next != nullptr) {
        tail = tail->next;
    }
    
    tail->next = new Node(value);
    return dummy.next;  // 返回真正的头节点
}
```
合并两个有序链表
```
// 使用哑节点（更简洁）
Node* mergeLists(Node* a, Node* b) {
    Node dummy(0);
    Node* tail = &dummy;
    
    while (a && b) {
        if (a->val <= b->val) {
            tail->next = a;
            a = a->next;
        } else {
            tail->next = b;
            b = b->next;
        }
        tail = tail->next;
    }
    tail->next = a ? a : b;
    return dummy.next;
}
```

```
#include<List.h>

// 方法1：反转链表后相加（推荐）
Node* getSum(Node* a, Node* b) {
    if (!a) return b;
    if (!b) return a;
    
    // 反转两个链表（从个位开始相加）
    auto reverseList = [](Node* head) -> Node* {
        Node* prev = nullptr;
        Node* curr = head;
        while (curr) {
            Node* next = curr->next;
            curr->next = prev;
            prev = curr;
            curr = next;
        }
        return prev;
    };
    
    Node* l1 = reverseList(a);
    Node* l2 = reverseList(b);
    
    Node* dummy = new Node(0);
    Node* current = dummy;
    int carry = 0;
    
    while (l1 || l2 || carry) {
        int sum = carry;
        if (l1) {
            sum += l1->value;
            l1 = l1->next;
        }
        if (l2) {
            sum += l2->value;
            l2 = l2->next;
        }
        
        carry = sum / 10;
        current->next = new Node(sum % 10);
        current = current->next;
    }
    
    // 反转结果链表恢复原始顺序
    Node* result = reverseList(dummy->next);
    delete dummy;
    return result;
}
```

不创建的合并有序链表
```
#include <iostream>
using namespace std;

Node* orderly_merge_list(Node* a, Node* b) {
    // 处理空链表情况
    if (!a) return b;
    if (!b) return a;
    
    // 确定头节点（较小的作为头）
    Node* head = nullptr;
    if (a->val <= b->val) {
        head = a;
        a = a->next;
    } else {
        head = b;
        b = b->next;
    }
    
    Node* current = head;
    
    // 合并两个链表
    while (a && b) {
        if (a->val <= b->val) {
            current->next = a;
            a = a->next;
        } else {
            current->next = b;
            b = b->next;
        }
        current = current->next;
    }
    
    // 连接剩余部分
    if (a) {
        current->next = a;
    } else {
        current->next = b;
    }
    
    return head;
}
```

# list
```
1. 基础增删改查（与 vector 类似，但性能不同）
函数	
push_back(val) / push_front(val)	在尾/头部插入元素	
pop_back() / pop_front()	删除尾/头部元素	
insert(pos, val)	在指定迭代器位置前插入元素	
erase(pos) / erase(first, last)	删除指定位置或区间元素	
front() / back()	获取头/尾元素的引用	O(1)
clear()	清空所有元素	O(n)
注意：list 没有 [] 运算符和 at() 函数。如果要访问中间元素，只能通过迭代器 ++ 一步步移动。

2. 容量与状态
函数	说明
empty()	判空
size()	返回元素个数（C++11 前为 O(n)，之后标准规定必须 O(1)）
resize(count)	重新指定大小，多删少补（默认补 0）
3. 独门绝技：链表专属特殊操作（面试高频）
这是 list 最精髓的部分，因为链表操作的是节点指针，这些函数都不涉及内存拷贝或移动，效率极高。

函数	说明	注意点
splice(pos, other)	将另一个链表 other 的所有节点“转移”到当前链表的 pos 位置。	other 会变为空。这是纯粹的指针重连，O(1) 复杂度，极快！
splice(pos, other, it)	将 other 中的单个节点 it 转移到当前位置。	O(1)，只需改变几个指针。
remove(val)	删除链表中所有等于 val 的元素。	不同于 erase，它是按值删除，复杂度 O(n)。
remove_if(pred)	删除所有满足条件（谓词）的元素。	例如：lst.remove_if([](int x){ return x > 5; });
unique()	移除连续重复的元素，只保留第一个。	通常需要先排序再调用，否则只能删相邻的重复项。
merge(other)	将另一个已排序的链表 other 合并到当前已排序的链表。合并后依然有序，且 other 清空。	前提是两个链表都必须有序，否则行为未定义。
sort()	对链表进行排序。	不能用 std::sort（因为需要随机迭代器），必须用 list 自带的 sort()，稳定排序，复杂度 O(n log n)。
reverse()	将链表完全反转。	O(n)，直接调换前后指针。
4. 迭代器与失效问题（非常重要）
list 提供的是双向迭代器（Bidirectional Iterator），支持 ++ 和 --，但不支持 it += 2。

引用/迭代器不会轻易失效：这是 list 的巨大优势。当你插入或删除元素时，其他元素的迭代器、指针、引用全都保持有效（只有被删除的那个迭代器会失效）。这与 vector 插入时可能全盘失效形成鲜明对比。

实战代码示例（一看就懂）
cpp
#include <iostream>
#include <list>
using namespace std;

int main() {
    list<int> l1 = {1, 2, 3};
    list<int> l2 = {10, 20, 30};

    // 1. 头插和尾插
    l1.push_front(0);   // {0,1,2,3}
    l1.push_back(4);    // {0,1,2,3,4}

    // 2. 重点：splice 拼接（将 l2 全部移到 l1 的尾部，l2清空）
    l1.splice(l1.end(), l2); 
    // l1: {0,1,2,3,4,10,20,30}
    // l2: {}

    // 3. 按值删除所有 2
    l1.remove(2);       // {0,1,3,4,10,20,30}

    // 4. 排序（合并和去重前必须排序）
    l1.sort();          // {0,1,3,4,10,20,30}

    // 5. 去重（只删相邻的重复项，这里没有重复，演示效果）
    l1.unique();        // 没变化

    // 6. 合并两个已排序的 list
    list<int> l3 = {5, 6, 15};
    l1.merge(l3);       // l1 和 l3 都必须已排序，合并后 l3 变空
    // l1: {0,1,3,4,5,6,10,15,20,30}

    // 7. 反转
    l1.reverse();       // {30,20,15,10,6,5,4,3,1,0}

    // 8. 遍历（只能用迭代器）
    for (int x : l1) {  // 范围for底层就是迭代器，完全支持
        cout << x << " ";
    }
    return 0;
}
```
