# 字符串操作
```
一、std::string::erase —— 删除字符
erase 用于从字符串中删除指定位置的字符，它有 三种重载形式：

1. erase(size_t pos = 0, size_t count = npos)
删除从位置 pos 开始的 count 个字符。
如果 count 缺省（或为 npos），则会一直删到字符串末尾。

string s = "Hello World";

s.erase(5, 6);   // 删除从索引5开始的6个字符 → "Hello"
// 索引5是空格，删掉 " World"

s = "ABCDEF";
s.erase(2);      // 只给 pos，缺省 count 为 npos → 从索引2删到末尾 → "AB"

⚠️ 重点记忆：
erase(pos) 不是“只删一个字符”，而是删掉从 pos 到结尾的所有字符，这是常见的误解，你在上一版代码里就栽在这里。

如果想只删一个字符，必须指定 count = 1：

s.erase(2, 1);   // 只删索引2这一个字符


2. erase(iterator position)
删除迭代器指向的单个字符。
返回被删除字符后一个字符的迭代器。

string s = "Hello";
auto it = s.begin() + 1;  // 指向 'e'
s.erase(it);              // 删除 'e' → "Hllo"
可以配合 std::find 等使用：

auto it = find(s.begin(), s.end(), 'l');
if (it != s.end())
    s.erase(it);   // 删除第一个 'l'



3. erase(iterator first, iterator last)
删除 [first, last) 区间内的所有字符。
返回 last 所指向的迭代器（删除后的位置）。

string s = "Hello World";
s.erase(s.begin() + 5, s.end());   // 删除从索引5到末尾 → "Hello"

s = "abcdef";
s.erase(s.begin() + 1, s.begin() + 4); // 删除 b,c,d → "aef"
常见误用提醒
erase(pos) 会删光后面所有，不是单字符删除。

下标 pos 必须合法（0 ≤ pos < size()），否则行为未定义。

对空串操作可能崩溃（如 s.erase(0) 没问题，但 s.erase(s.length()-1) 在空串时越界）。

二、std::string::substr —— 截取子串
substr 用于获取一个子串，原字符串不变，它返回一个新的 string。

string substr(size_t pos = 0, size_t count = npos) const;
参数
pos：开始位置（索引从 0 开始）。

count：要截取的字符数。
如果 count 缺省或超过 size() - pos，则截取到字符串末尾。

string s = "Hello World";

string sub1 = s.substr(0, 5);    // "Hello"
string sub2 = s.substr(6);       // 从索引6到末尾 → "World"
string sub3 = s.substr(6, 2);    // "Wo"
string sub4 = s.substr(6, 100);  // count 超过剩余长度，得到 "World"

结合去前导零的使用（删数问题常见）

string s = "0013";
int start = 0;
while (start < s.size() - 1 && s[start] == '0') start++;
string result = s.substr(start); // "13"
注意要保留至少一个字符，防止结果为空串。
```
