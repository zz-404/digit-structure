**find 函数的工作原理非常简单：它从 first 开始，逐个遍历序列中的每个元素，直到 last（但不包括 last），将每个元素与 value 进行比较。比较使用的是 operator==。因此，你查找的类型必须支持 == 操作符，或者你可以为自定义类型重载该操作符，find函数支持使用索引查找，也可以用于string字符串！！！**
